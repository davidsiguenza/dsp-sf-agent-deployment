# dsp-sf-agent-deployment (plugin)

This plugin contains a single skill, `dsp-sf-agent-deployment`, under `skills/`.

The skill is a procedural runbook for connecting an existing Agentforce agent to a Storefront Next storefront via MIAW. See [`skills/dsp-sf-agent-deployment/SKILL.md`](./skills/dsp-sf-agent-deployment/SKILL.md).
